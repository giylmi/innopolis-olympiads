Открытая олимпиада

Открытая олимпиада школьников по программированию Университета Иннополис.

Информация о заочном этапе

Заочный этап олимпиады проводится в один тур, который состоится 22 марта 2015 года. Начало тура запланировано на 11:00, продолжительность – 5 часов. По результатам заочного тура лучшие школьники будут приглашены для участия в очном этапе. Точный список приглашенных будет опубликован на сайте после подведения итогов отборочного этапа.

Заочный этап проводится с использованием сети Интернет. Школьники могут участвовать на любых компьютерах. Для проверки решений будет использована система PCMS2.

Заочный этап будет проходить по правилам Всероссийской олимпиады школьников по информатике. На туре Вам будут предложены 4-5 задач, решения которых нужно отправить в жюри через веб-интерфейс сайта олимпиады. Все попытки будут проверяться в автоматическом режиме.

Правила

Решением является программа, написанная на одном из языков программирования:

  • C++
  • C
  • Java
  • Pascal
  • Python

При компиляции текста программы, которую участник сдает на проверку, жюри использует вполне определенные командные строки для компиляции решений:

Университет Иннополис
